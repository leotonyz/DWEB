Trilha Do aprendizado

A matéria de desenvolvimento web tem como foco ensinar como se cria um webiste, todas os processos, detalhadamente, para que assim o aluno consiga sozinho desenvolver o conteúdo, a estilização e a interação do usuário com a página.
Inicialmente, aprendemos a manipular o HTML, seus códigos e maneiras de utilização, após isso aprenmos a fazer a estilização do conteúdo através do CSS e por fim aprendemos o JavaScript, que serve para gerar uma maior interação do site e suas funções com o usuário, e não apenas imprimir a imagem e textos para o usuário.

Sobre o projeto

O projeto "Trilha do Aprendizado", foi desenvolvido como uma espécie de história do conteúdo, onde a cada aula e exercício nós fazíamos o upload do arquivo referente a esse conteúdo. Isso é extremamente importante, pois assim conseguimos desenvolver uma melhor percepção do que já foi feito, podemos voltar para conteúdos antigos e revisar, e também deixamos registrado nossa jornada.
