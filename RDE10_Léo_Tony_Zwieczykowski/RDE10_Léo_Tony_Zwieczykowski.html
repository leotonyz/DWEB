CSS CASCADING STYLE SHEETS

O CSS - Cascading Style Sheets - tem a finalidade de fazer a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, espaçamentos, entre outros.

Sintaxe:

Há tres formas de utilizar o CSS

Inline

  1. Os códigos são definidos dentro de uma tag HTML
  2. Afetam somente a tag em que foi definida

Incorporado
  1. Os códigos CSS são definidos no cabeçalho, dentro da tag head.
  2. Afetam somente documentos em que foram definidos

Externo
  1. Os códigos CSS são definidos em um arquivo externo
  2. Afetam todos os documentos que estão linkados ao CSS
